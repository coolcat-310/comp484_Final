Hardware Dependencies

  1. Raspberry Pi (Original Model)
  2. Tiny Breadboard (Optional)
  3. Pi Breakout Cobbler (Optional)
  4. 8Gb SD Card
  5. Wireless dongle
  6. Adafruit 9DOF BNO055 Sensor
  7. 10 Jumper Wires
  8. Soldering Iron w/solder

Software Dependencies

  1. Raspberry Pi (Original Model)
  2. Python w/Flask
  3. SSH & FTP (not really software but make sure your linux disc can do both protocols
  4. Putty
  5. Motion Fun Respository
  6. Blender
  7. A webGL enabled browser (google chrome is fine)
